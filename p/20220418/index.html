2022年浏览器使用Flash的两种方法

发表于 2022-04-18 更新于 2022-12-17 分类于 实用 阅读次数：
本文字数： 1.1k 阅读时长 ≈ 1 分钟

Adobe公司的flash早已被淘汰，但是某橙特供版还在更新，目前为34，but弹窗泛滥，所以这也是你为什么会来到这的原因。

方法一：使用自带Flash的浏览器

（tips：下载地址在文章末尾）

由于早期的chrome和Firefox自带flash插件，不需要额外下载flash插件，所以我们可以下载旧版本浏览器来达到使用flash的目的。在网上查找得知，chrome53是最后一个内置flash的版本(flash版本22)，chrome68是最后一个默认允许运行flash的版本，chrome88开始禁用flash；Firefox可自行查找.

  • 优点

  1. 自带Flash，虽然版本旧，但是功能依然可用

  2. 避免重橙特供版的广告弹窗

  3. 相对于支持Flash的旧版本浏览器来说，省去下载Flash的过程

  4. 与已安装版本不冲突

  • 缺点

  1. Chrome版本旧，体验可能不如新版本

  2. 数据保存在Data目录下，不方便数据同步

习惯Chrome所以没有制作Firefox的，详细制作可参考自己制作Chrome便携版实现多版本共存，或自行百度教程，基本上都是类似的方法。

方法二：安装Flash Player，并使用支持Flash的浏览器

Tips：安装Flash前，电脑中如果已经安装了其他版本Flash，先使用“Adobe_Flash_Player一剑全清__V4.exe”清除电脑上的Flash，重启电脑后，再安装Flash。

  1. 安装Falsh（注：安装Clean Flash Player需要 .NET 4.0）。

点击 CleanFlash-34.0.0.192_Installer .exe ，勾选，点击AGREE

选择需要安装的类型（Chrome需要勾选ppapi，Firefox需要勾选npapi，IE需要勾选ActiveX），默认全部勾选，点击下一步 NEXT

这里是安装独立的Flash Player，就是本地播放Flash文件的应用，看个人需求，然后点击NEXT完成安装

  1. 浏览器。上面介绍过，自Chrome 88起禁用Flash，所以可以使用低于88的版本，例如Chrome 87。这里提供两种版本，一个是离线安装版，一个是便携版。如果你正在使用Chrome并且版本高于87，建议使用便携版；如果你没有使用Chrome，安装版和便携版均可。
