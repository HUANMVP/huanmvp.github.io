内置Flash的Chrome浏览器

发表于 2022-04-18 分类于 实用 阅读次数：
本文字数： 285 阅读时长 ≈ 1 分钟

Chrome53，内置Flash22版本，应该是最后一个内置Flash插件的Chrome版本。

自测4399小游戏可用，其他网站自测。网页字小缩放可以设置成125%或更大。

  • 优点

  1. 自带Flash，虽然版本旧，但是功能依然可用

  2. 避免重橙特供版的广告弹窗

  3. 相对于支持Flash的旧版本浏览器来说，省去下载Flash的过程

  4. 与已安装版本不冲突

  • 缺点

  1. Chrome版本旧，体验可能不如新版本

  2. 数据保存在Data目录下，不方便数据同步

习惯Chrome所以没有制作Firefox的，详细制作可参考自己制作Chrome便携版实现多版本共存，或自行百度教程，基本上都是类似的方法。
